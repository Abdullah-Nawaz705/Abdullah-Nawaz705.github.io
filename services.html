Accessibility Consulting Services

I provide comprehensive accessibility testing and consulting services to help organizations create truly inclusive digital experiences. My approach combines technical expertise with real-world user experience from over 10 years of daily assistive technology use.

Manual Screen Reader Testing

What automated tools miss, I catch. While automated scanners detect about 30% of accessibility issues, I identify the remaining 70% through comprehensive manual testing with real assistive technologies.

My Testing Process:

  • Complete navigation flow analysis using NVDA on Windows and VoiceOver on iOS
  • Form interaction and error handling assessment
  • Content structure and heading hierarchy evaluation
  • Focus management and keyboard navigation testing
  • Interactive element usability verification

Ideal for: Websites, web applications, and digital platforms requiring authentic user experience validation.

WCAG 2.1/2.2 Compliance Audits

Comprehensive audits ensuring your digital products meet WCAG Level AA standards, with AAA recommendations where feasible.

Audit Components:

  • Detailed compliance mapping to WCAG success criteria
  • Prioritized issue identification with impact assessment
  • Specific remediation recommendations with code examples
  • Implementation timeline and resource planning guidance
  • Follow-up verification testing

Deliverable: Comprehensive report with actionable recommendations and compliance roadmap.

Mobile App Accessibility Testing

Specialized testing for iOS applications using VoiceOver to ensure optimal accessibility and usability.

Mobile Testing Focus:

  • VoiceOver (iOS) compatibility and usability testing
  • Touch gesture accessibility and navigation patterns
  • Dynamic content and state change announcements
  • Custom control accessibility implementation
  • Real-world usage scenario validation

Perfect for: iOS native apps, progressive web apps, and mobile web applications.

Document Accessibility Review

Ensuring your digital documents are accessible to screen reader users and comply with accessibility standards.

Document Types:

  • PDF accessibility and remediation guidance
  • Microsoft Office document accessibility (Word, Excel, PowerPoint)
  • Web-based document accessibility
  • Alternative format recommendations

Outcome: Documents that work seamlessly with assistive technologies and provide equal access to information.
